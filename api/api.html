read one

Description
Given an username, returns matching username's info in JSON format.

HTTP request
GET http://localhost/is216Project/api/talent/readOne.php?username=selena
where username is the username parameter, and
selena is the username value.
In the case of local API installation, the exact URL may vary - please check your local setup.

search

Description
Returns a collection of stars matching the search query in JSON format.

HTTP request

  • Search all user by Gender && Year of experience && Talent
    GET http://localhost/is216Project/api/talent/search.php?g=m&y=10&t=fishing
    where g is the gender parameter, y is the year of experience (in range) and
    f (or F) indicates female
    m (or M) indicates male

  • Search all user by Gender && Year of experience
    GET http://localhost/is216Project/api/talent/search.php?g=m&y=10
    where g is the gender parameter, and
    f (or F) indicates female
    m (or M) indicates male
    and y is the year of experience (in range)

  • Search all user by Gender && Talent
    GET http://localhost/is216Project/api/talent/search.php?y=10&t=fishing
    where g is the gender parameter, and
    f (or F) indicates female
    m (or M) indicates male
    and t is the Talent

  • Search all user by Gender
    GET http://localhost/is216Project/api/talent/search.php?g=m
    where g is the gender parameter, and
    f (or F) indicates female
    m (or M) indicates male

  • Search all user by Year of experience
    GET http://localhost/is216Project/api/talent/search.php?y=10
    where y is the year of experience which is in range, eg. 10 = min 1, max 10 years

  • Search all user by Talent
    GET http://localhost/is216Project/api/talent/search.php?t=music
    where t is the Talent

  • In the case of local API installation, the exact URL may vary - please check your local setup.
